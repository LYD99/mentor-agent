# 背景和痛点
1、苦寻导师不得。我一直在寻找某个导师，能教我怎么成为一个厉害的人，但是总是没法遇到
2、寻道不得。学习的时候总感觉自己的效率不高，也很难坚持下来，如果有一个导师指导我一步步成长，或许就能很快成为某个领域杰出的人才，而不是像现在这样，觉得碌碌无为。
3、晚上想千条路，醒来走原路。总是觉得有目标，但是不知道如果脚踏实地去落实，一直浮在表面，总是动不动因为胸怀大志，却无从下手而感到迷茫，感到焦虑。如果有个导师可以根据我的目标，制定详细的计划，一步步指导我，那变得优秀便指日可待。
4、似懂非懂。学到一个知识，感觉好像懂了，但是一实践发现压根不懂，又或是过了一段时间全忘了，仿佛没学过，很难做到知行合一，因为只有很好的知，才知道哪些行是有效加深知的动作。如果有这么一个资深的导师，提出一些实践，哪怕是一些题目，能够让自己去实践，学知识就不会再是那么浅的了。
5、零散的认知。到处看，看一个信一个，知识十分零散，根本不成体系，一问细节，一问本质，就懵了，一遇到相关领域的问题，就卡壳了。很难有系统化的认知。如果有个导师可以给领域系统化的知识，并制定如何一步步把系统化的知识技能点亮的话，在该知识领域，就会有充足的底气，也就不容易在遇到打击时，感到焦虑和挫败。

# 目标
我打算打造一个导师agent，帮助所有希望能快速成长，但又困于没有“贵人”教导的普通人。
立志于解决以上提到的痛点：
1、无人教（支持AI导师和真人导师，用户可选择不同风格的mentor）
2、无成长地图（成长地图分阶段：入门-进阶-精通，支持动态调整和个性化定制）
3、没有具体步骤（每阶段有具体目标、任务、反馈方式，支持目标拆解）
4、没有实践指导与反馈（每个知识点配有基础练习、项目实战、反思总结，自动生成小测验和实践建议，支持自动批改/点评）
5、知识不成系统（知识地图可视化，辅助系统化认知，用户可一眼看到自己的知识结构）

# 方案
1、产品形态
网页UI： 以网页的形式展示
2、功能
- chat with agent：mentor agent（教导，支持个性化风格如严师型、鼓励型、陪伴型）、plan agent（成长地图制定与动态调整）、knowledge agent（辅助系统化认知，知识地图可视化）、exercise agent（实践指导与自动点评）。可以与agent进行单独的聊天
- 各种bot通信：参考openclaw，支持对接各个通讯软件，实现人机交互
- 成长地图：agent会 按照用户的成长诉求，定制成长地图，并支持用户修改，成长地图分阶段，适配不同用户画像（如学生、职场新人、转行者等）
- 定时任务：依据成长地图上的计划，生成定时任务，准时启动
- 日报与回顾：用户自己提交日报，系统自动生成日报，并支持每周/每月回顾，帮助用户看到成长曲线
- 成就系统：增加成长徽章、成就激励，提升用户持续实践的动力
- 社区与互助：支持用户互评、互助，形成学伴或成长小组，也可引入达人答疑/真人导师补充AI mentor
- 数据安全与隐私：明确用户数据归属和隐私保护，尤其是成长记录、日报等敏感信息
- 商业化与可持续性：部分功能免费，进阶功能/真人导师/定制成长方案可收费

---
力求在“个性化成长路径+实践反馈+社区激励”三方面做深做透，形成闭环，用户粘性会更强！
